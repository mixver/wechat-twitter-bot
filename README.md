## Wechat-twitter-bot

Twitter图片、视频下载微信机器人

## 配置

    cp .env.example .env

TWITTER_TOKEN       twitter的登录token 

## 运行

    npm install
    ts-node main.ts

## 使用

发送 。推特图 [推特分享链接]  给机器人，即可下载twitter上分享的图片或视频。
